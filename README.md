# 🎭 Soumil | Acting Portfolio

A personal acting portfolio website built with React and Vite, showcasing work, experience, and presence as an actor.

🔗 **Live Site:** [acting-portfolio-nine.vercel.app](https://acting-portfolio-nine.vercel.app)

---

## Tech Stack

- **React 19** — UI framework
- **Vite** — Fast build tool and dev server
- **React Router DOM** — Client-side routing
- **Tailwind CSS** — Utility-first styling
- **Matter.js** — Physics engine (for interactive UI elements)
- **Vercel** — Deployment and hosting

---

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/soumil007-pdg/acting-portfolio.git
cd acting-portfolio

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```

The output will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project is deployed on **Vercel**. Any push to the `main` branch automatically triggers a new deployment.

To deploy your own fork:

1. Import the repository on [vercel.com](https://vercel.com)
2. Vercel will auto-detect the Vite framework
3. Hit **Deploy** — no additional configuration needed
